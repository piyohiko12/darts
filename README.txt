Darts Motion Coach — PWA版 セットアップ手順
=================================================

このフォルダの中身（すべて同じ階層に置いてください）:
  index.html              … アプリ本体
  manifest.webmanifest    … PWA設定
  sw.js                   … オフライン用サービスワーカー
  icon-192.png            … アイコン
  icon-512.png            … アイコン
  apple-touch-icon.png    … iOS用アイコン

------------------------------------------------
重要：PWA（ホーム画面に追加・オフライン動作）は
「ファイルを直接開く（file://）」では動きません。
HTTPSで配信して初めてインストール／オフラインが有効になります。
（ローカルで index.html を開いてもアプリ自体は動きますが、
 PWA機能だけ無効になります。）
------------------------------------------------

■ いちばん簡単な配信方法（無料・登録ほぼ不要）
1. https://app.netlify.com/drop を開く
2. このフォルダごと（zipを展開した中身）をブラウザにドラッグ＆ドロップ
3. 数秒で「https://〜.netlify.app」のURLが発行される
4. そのURLをスマホのブラウザ（Chrome/Safari）で開く

■ ホーム画面に追加（インストール）
・Android(Chrome)：メニュー（⋮）→「アプリをインストール」/「ホーム画面に追加」
・iPhone(Safari)：共有ボタン（□↑）→「ホーム画面に追加」

■ 他の配信先（どれでもOK）
・GitHub Pages：リポジトリに6ファイルを置き、Settings→Pagesで公開
・Vercel / Cloudflare Pages：このフォルダをデプロイ
・自宅サーバ：HTTPSで配信（localhostでも可：開発時）

■ オフラインについて
初回はオンラインで一度「分析を開始」まで実行してください。
そのときAIモデル等がキャッシュされ、以降はオフラインでも分析できます。
（モデル更新時はキャッシュ名 dmc-v1 を sw.js で上げると更新されます）

■ 使い方
真横・スロー・複数本のスロー動画を選び「分析を開始」。
総評→映像→比較→詳細→履歴 のタブで確認できます。
数値やスコア、スロー自動検出は2D姿勢推定による「目安」です。

動画は端末内で処理され、外部に送信されません。
